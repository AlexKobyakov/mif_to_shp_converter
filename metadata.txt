about=🌟 Professional plugin for converting MapInfo Interchange Format (MIF/MID) and MapInfo TAB files to ESRI Shapefile or GeoJSON format with comprehensive multilingual support and modular architecture.
    
    🚀 Key Features:
    • 🏗️ Modular architecture with separated GUI components
    • 📁 Support for both MIF/MID and TAB input formats
    • 🌐 Export to Shapefile (.shp) or GeoJSON (.geojson) formats
    • 📄 Single file or batch folder processing
    • ⚡ Multithreading support (1-16 threads) for faster processing
    • 🌍 Coordinate system specification (EPSG, PROJ4, WKT)
    • 📊 Detailed progress tracking with modern progress bar
    • 📋 Comprehensive logging with color-coded messages
    • 📈 Results table with file-by-file status tracking
    • 🌐 Multilingual interface supporting 9 major world languages
    • 🎨 Modern, professional UI design with icons and gradients
    • ✅ Auto-add converted layers to QGIS project
    • 📖 Built-in CRS examples and help system
    • 🛡️ Safe cancellation of running processes
    • 📊 Conversion statistics and completion notifications
    • 🔄 RTL (Right-to-Left) support for Arabic language
    
    🏗️ Architecture Improvements:
    • gui/ package with modular components
    • translations.py - Enhanced multilingual system
    • worker.py - Separated processing logic
    • Improved code maintainability and extensibility
    
    🌐 Supported Languages (9 languages):
    • 🇷🇺 Русский (Russian) - Complete translation
    • 🇺🇸 English - Complete translation
    • 🇨🇳 中文 (Chinese) - Complete translation
    • 🇮🇳 हिंदी (Hindi) - Complete translation
    • 🇪🇸 Español (Spanish) - Complete translation
    • 🇸🇦 العربية (Arabic) - Complete translation with RTL support
    • 🇫🇷 Français (French) - Complete translation
    • 🇧🇷 Português (Portuguese) - Complete translation
    • 🇩🇪 Deutsch (German) - Complete translation
